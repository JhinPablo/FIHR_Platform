# Local Demo Dataset

Dataset pequeno, versionable y sin PHI real. Imita la forma de MIMIC-IV y
MIMIC-CXR-JPG para probar el flujo local: `Patient`, `Observation`, `Media`,
`DiagnosticReport`, MinIO y RiskReport.

Los datasets reales deben descargarse fuera del repositorio:

```text
project2/datasets/mimic-iv/
project2/datasets/mimic-cxr-jpg/
```

Esas carpetas siguen ignoradas por Git.
